# Strapi

This template helps you deploy a Strapi app. Under the hood it creates one service.

## App Service

This service is based on the official Strapi Docker image.

- Docker Image
  - Name: [`strapi/strapi`](https://hub.docker.com/r/strapi/strapi)
  - Version: `latest`
- Volumes
  - `/srv/app`
